teiid-tools
===========

This project contains tools to help with Teiid development tasks

-------
Install Project
-------

Get project:
-   run: git clone git@git@github.com:teiid/teiid-tools.git

Build project (install archetypes):

-   run:  mvn -s ./settings clean install
